Location Tools

Find a location using New Zealand Transverse Mercator (NZTM) X & Y coordinates.



Follow the below steps to use the Location Tools:
  1. Click the Location Tools button on the tool bar at the top of the map.
  2. Enter your NZTM coordinates
    E.g. X: 1570300 Y: 5180600
    The input boxes will only allow numbers and 1 decimal point to be entered.
  3. Click/press the Locate Coordinates button.
    The sidebar will closed when this button is clicked.
  4. Click/press the Clear button to remove the coordinate location graphic and popup from the map.

This functionality is available when viewed using a desktop and tablet computer. It is not available when using a smart phone.
