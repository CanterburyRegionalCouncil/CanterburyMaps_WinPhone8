Printing

Create a print ready file of your map.



Follow the below steps to Print your map:
  1. Choose a map template from the Map Template drop down list.
  2. Choose an output file format from the Output Type drop down list.
  3. Choose the output file image quality from the Image Quality drop down list.
  4. Enter Map Title & Author Title text into the available fields then click/tap .
  5. Once the output file has been created click/tap to preview, print and/or save.
