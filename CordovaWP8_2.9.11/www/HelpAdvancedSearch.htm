Advanced Search

Select features by their attributes using a Query Expression.



Follow the below steps to construct a Query Expression:
  1. Choose a layer to query for features from the Active Layer drop down list.
    Note: Only layers that are present in the map will appear in this list.



  2. Choose from the Field: drop down list the attribute column name to search for values in.
    E.g. To find a road choose;
    Active Layer: Road labels
    Field: name

  3. Choose an Operator: from the drop down list.



  4. Enter the value to search for into Sample Values
    Click/tap to get a sample of values in the chosen field that are present in the current map extent.

  5. Click/tap Add Expression then Start Search

  6. Query Expression Examples:
    (Active Layer - expression)

    1. Land Parcels - TA_Property_ID = '21251-08700 '

    2. Consented Activities - Lapses < '1992-10-05T00:00:00.000Z'

    3. Camping Site - DATE_SUPPLIED > '2013-02-22T00:00:00.000Z'

    4. Parks, forests and reserves labels - PLACENAME LIKE 'Pudding%'

    5. Consented Activities - ConsentNo IN ('CRC905348.2','CRC040177')

  7. Query results appear in the Results Table at the bottom of the screen.
    From the Results Table;
    • Single click/tap records in the results table to highlight these in the map.
      Hold Ctrl & click selected records to highlight multiple records in the results table.

    • Click/tap to zoom to the extent of selected records.

    • Double click records in the results table to zoom to these features in the map.

    • Click/tap to buffer selected features.
      Active Layer features intersecting the buffer can be selected by right-clicking the buffer and choosing Select Features in Active Layer.



    • Click/tap to save the selected records to a csv file.

    • Click/tap to unselect the highlighted records.

    • Click/tap to remove highlighted records from the results table.

    • Click/tap to remove all records from the results table.
