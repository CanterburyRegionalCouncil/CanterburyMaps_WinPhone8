Map Gallery

Switching between different maps from the Canterbury Maps map gallery.



Follow the below steps to use the map gallery:
  1. If already viewing a map, click/tab the map gallery icon from the main toolbar in the map.

  2. Click/tap one of the Map Group options to view its map list.

  3. Click/tap a map in the map list to open this map.
    Note this will close your current map.

    Use the map list page selector at the bottom of the list to view all maps in groups containing more than 6 maps.


  4. Click/tap when viewing the map list to go back to Map Groups.

  5. Click/tap (top right corner) to close the Map Gallery without changing map.
